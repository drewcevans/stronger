# Radically Simple Strength — Intermediate Template B

4-day program rotating primary and secondary versions of the four main barbell lifts, plus assistance work.

## Structure

Each workout has three sections:

1. **Primary lift** — the heavy main lift for the day
2. **Secondary lift** — a lighter version of a different main lift (100% = 85% of the paired primary workout's top set)
3. **Assistance** — accessory exercises

| Workout | Primary      | Secondary            | Assistance                | Secondary weight               |
| ------- | ------------ | -------------------- | ------------------------- | ------------------------------ |
| A       | Bench Press  | Press (secondary)    | Skull Crusher             | 85% of Workout C Press top     |
| B       | Squat        | Deadlift (secondary) | Chin-up                   | 85% of Workout D Deadlift top  |
| C       | Press        | Bench (secondary)    | Bicep Curl, Lateral Raise | 85% of Workout A Bench top     |
| D       | Deadlift     | Squat (secondary)    | Dumbbell Row              | 75% of Workout B Squat top     |

## Workouts

### Workout A — Bench Press

#### Primary: Bench Press

| Set         | Reps   |
| ----------- | ------ |
| Bar × 10    | warmup |
| 45% × 5    | warmup |
| 65% × 3    | warmup |
| 85% × 2    | warmup |
| 100% × 3–5 | top set |
| 85% × 5–8+ | backoff |

> **Top set progression**: Minimum 3 reps. If 5 reps are completed, increase top set weight by **2.5 lbs** the following week.
>
> **Backoff set progression**: Do as many reps as possible, minimum 5, goal of 8. If 8 reps are completed, increase backoff weight by **2.5 lbs** the following week. Top set and backoff weights are tracked independently.
